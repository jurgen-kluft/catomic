Detailed Description

template<typename T = u32>
class xcore::atomic::ring< T >

Ring buffer. Simple and efficient lock free circular fifo. Thread safety is guarantied only for the 'single writer -> single reader' case. Ring size is restricted to the powers of two. One item is always reserved for internal operation.
