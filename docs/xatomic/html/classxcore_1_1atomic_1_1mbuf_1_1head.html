Detailed Description

mbuf head which can be used to share memory between different levels of a protocol stack. Allows for headers to be added and removed in such a way that extra copying of memory is avoided.

Constructor & Destructor Documentation

xcore::atomic::mbuf::head::head ( void  ) [inline, protected]

Constructor function. Needs to be private since it should not be used. Only allocators should be used to create new mbufs.

xcore::atomic::mbuf::head::~head ( ) [inline, protected]

Destructor function. Again this needs to be private since it should not be called. In its place free() should be used.
