Detailed Description

List node. Nodes can be added to the list or chained.

See also:
head

Chain is basically singly linked / null terminated list. Users are expected to do all the hard-work. ie initialize, set, clear, null-terminate, etc. However some of the list methods are chain friendly, chop() for example, for efficiency. Node can be either in a chain or in a list, not both.

See also:
chop()
